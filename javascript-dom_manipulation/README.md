# JavaScript DOM Manipulation Project

This project contains JavaScript scripts that manipulate the DOM in various ways. It demonstrates the following tasks:

## Tasks

1. **Color Me**
   Update the text color of the header element to red using `document.querySelector`.

2. **Click and Turn Red**
   Change the header text color to red when the element with `id="red_header"` is clicked.

3. **Add `.red` Class**
   Add the class `red` to the header element when the element with `id="red_header"` is clicked.

4. **Toggle Classes**
   Toggle the header class between `red` and `green` when the element with `id="toggle_header"` is clicked.

5. **List of Elements**
   Add a `<li>` element with text "Item" to a `<ul>` with class `my_list` when the element with `id="add_item"` is clicked.

6. **Change the Text**
   Update the header text to "New Header!!!" when the element with `id="update_header"` is clicked.

7. **Star Wars Character**
   Fetch a Star Wars character from the API and display its name in the element with `id="character"`.

8. **Star Wars Movies**
   Fetch all Star Wars movies from the API and list their titles in a `<ul>` with `id="list_movies"`.

9. **Say Hello!**
   Fetch a translation of "hello" in French and display it in the element with `id="hello"`.

## Requirements

- Use only `document.querySelector` for DOM selection.
- Scripts must be semistandard compliant.
- HTML should not reload for any DOM updates or fetch requests.
- All scripts are tested to run in Chrome (version 57.0 or later).
